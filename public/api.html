Moduł

Podczas tworzenia strony, został do niej wykonany autorski moduł nodejs. Od teraz każdy może go używać do woli w swoich projektach!

Instalacja:

W konsoli, która jest ustawiona na twój projekt nodejs, wpisz polecenie: 
npm i santatracker
Lub jeśli używasz Yarn, użyj polecenia: 
yarn add sanatracker

Przykładowe użycie:

const santatracker = require('santatracker');
console.log(`Do wigilii zostalo ${santatracker.days} dni, ${santatracker.hours} godzin, ${santatracker.minutes} minut i ${santatracker.seconds} sekund`);
